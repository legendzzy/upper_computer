# upper_computer
#### 默认窗口
* 选择导入，将仪器的txt文件的历史记录选中。选中后会以表的形式在主界面中显示数据。  
* 选择起始时间和终止时间，限定数据的时间范围。默认的起始终止时间为文件中数据的第一条和最后一条。  
* 选择对象，默认为all，即所有气体，此时不能选择上下限；可单独选中某气体，并给定数值上下限，此时会以该气体的数值范围进行数据行筛选。  
* 限定筛选条件后点击筛选按钮，表中将会显示符合条件的数据行。  
* 点击复位，数据表会返回为文件数据的初始状态。  
* 点击统计图显示，将会以折线图形式显示此时数据表中的数据。  
* 点击报警分析，进入报警分析模块。
#### 统计图显示窗口
* 在图表界面，使用鼠标滚轮可以操作图表的缩放。可以使用折线图下方的滚动条对折线图按时间滚动。  
* 在图表界面左上方，有显示曲线的选框，若上一步未选择某项气体，则默认全部选中，以显示所有折线，否则默认之选中当前气体。勾选中的气体会显示对应折线图。  
* 鼠标左键单击折线图中的点或线段，将在界面右下方的点信息中显示距离鼠标点击最近的点的信息，包括所属曲线、对应时间、对应数值，并将该点标红显示。再次点击该点，该点恢复原状。
* 鼠标放置于折线图中的点或线段上，会显示悬浮提示，标出折线名称以及该折线中距离鼠标最近的点的信息。
* 鼠标在折线图中，会有x轴的游标自动跟随。界面左下角会显示出当前选中气体在该时刻的数值（如果存在），界面左边框中会显示该时刻所有气体的数值，并标有绿橙红三种颜色，分别代表小于低报，低报以及高报。
#### 报警分析窗口
* 在左上角选框中选择气体对象，会显示出该气体的属性信息。
* 选择筛选条件，点击确定，将会以表格形式显示该时间段内符合报警条件的时间段。该表格包括每条时间段的起止时间、最大值、平均值、持续时间。左下角会显示查询到的记录的条数。
* 选中一行，点击统计图显示按钮，则会以用统计图显示窗口展示该段时间段的报警数据。
