# upper_computer
#### 默认窗口
* 选择导入，将仪表的历史记录txt文件选中。选中后会以数据表的形式在主界面中显示。  
* 选择起始时间和终止时间，限定数据的时间范围。默认的起始终止时间为文件中数据的第一条和最后一条。  
* 选择对象，默认为all，即所有气体，此时不能选择数值上下限；可单独选中某气体，并给定数值上下限，此时会以该气体的数值范围进行数据行筛选。  
* 限定筛选条件后点击筛选按钮，表中将会显示符合条件的数据行。  
* 点击复位，数据表会返回为文件数据的初始状态。  
* 点击统计图显示，将会启动统计图显示窗口，主要以折线图形式显示此时数据表中的数据。  
* 点击报警分析，进入报警分析模块。
#### 统计图显示窗口（此窗口与主窗口的筛选数据结果绑定）
* 在图表界面，使用鼠标滚轮可以操作图表的缩放。可以使用折线图下方的滚动条对折线图按时间滚动。
* 滚动条的左右键按钮可以使图表直接滚动到最左/右端
* 滚动条的最左侧按钮为完整显示图表按钮，点击之后图表界面会完整的显示该曲线图。注意此时，鼠标滚轮功能失效，需要点击左下角的**刷新**按钮，方可还原到初始状态。
* 在图表界面左上方，有选择显示曲线的选框，若上一步未选择某项气体，则默认全部选中，以显示所有折线，否则默认选中当前气体。勾选中的气体会显示对应折线图。  
* 鼠标左键单击折线图中的点或线段，将在界面右下方的浓度点信息中显示距离鼠标点击最近的点的信息，包括所属曲线、对应时间、对应数值，并将该点标红显示。再次点击该点，该点恢复原状。
* 鼠标放置于折线图中的点或线段上，会显示悬浮提示，标出折线名称以及该折线中距离鼠标最近的点的信息。
* 鼠标在折线图中，会有垂直于x轴的游标自动跟随。界面左下角的时刻浓度信息栏会显示出当前选中气体在该时刻的数值，界面左下的框中会显示该时刻所有气体的数值，并标有绿橙红三种颜色，分别代表小于低报，高于低报低于高报以及高于高报。（氧气的报警策略略有不同）
#### 报警分析窗口（此窗口独立）
* 在左上角选框中选择气体对象，会显示出该气体的属性信息。
* 选择筛选条件，点击确定，将会以表格形式显示该时间段内符合报警条件的时间段。该表格包括每条时间段的起止时间、最大值、平均值、持续时间。左下角会显示查询到的记录的条数。
* 选中一行，点击统计图显示按钮，则会以统计图显示窗口展示该段时间段的报警数据。

**数据表可点击某一列头，将表格按此列进行从高到低/从低到高排列**
***
#### 读入文件格式说明
* 历史记录文件格式为txt
* 文件头部需要记录气体信息，一行对应一种气体，不超过6种
* 每一行需要记录如下信息：气体名称、量程、单位、低报值、高报值， 每个信息之间用若干空格隔开
* 文件头部与后续数据记录以若干空行隔开
* 数据记录包括日期、时间和各个气体的读数，一条记录为一行，每一列之间以若干空格隔开
