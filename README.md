# upper_computer
## 导入txt文件显示数据功能比较完善  
* 选择导入，将仪器的txt文件的历史记录选中。选中后会以表的形式在主界面中显示数据  
* 选择起始时间和终止时间，限定数据的时间范围。默认的起始终止时间为文件中数据的第一条和最后一条  
* 选择对象，默认为all，即所有气体，此时不能选择上下限；可单独选中某气体，并给定数值上下限，此时会以该气体的数值范围进行数据行筛选  
* 限定筛选条件后点击筛选按钮，表中将会显示符合条件的数据行  
* 点击复位，数据表会返回为文件数据的初始状态  
* 点击图表显示，将会以折线图形式显示此时数据表中的数据  
* 在图表界面，使用鼠标滚轮可以操作图表的缩放。可以使用图表下方的滚动条对图表按时间滚动  
* 在图表界面右上方，有显示曲线的选框，默认全部选中，以显示所有折线。勾选中的气体会显示对应折线图  
* 鼠标左键单击折线图中的点或线段，将在界面下方的点信息中显示点击最近的点的信息，包括所属曲线、对应时间、对应数值，并将该点标红显示。再次点击该点，该点恢复原状
